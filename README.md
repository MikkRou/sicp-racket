# Sicp Racket Exercises

### Requirements

- Racket
- Rackunit (`raco pkg install rackunit`)
- [SICP Support for DrRacket](https://docs.racket-lang.org/sicp-manual/index.html) (`raco pkg install sicp`)
- Make

### Using

run `make test`
